= helidon-mongodb-cqrs

Sample Helidon MP project that includes multiple REST operations.

== Build and run


With JDK21

[source,bash]
----
mvn package
java -jar target/helidon-mongodb-events.jar
----

== Running the Scenario with cURL

=== 1. Create a Card

[source,bash]
----
curl -X POST http://localhost:8080/cards
----

=== 2. Fetch a Card by ID

[source,bash]
----
curl http://localhost:8080/cards/{cardId}
----

=== 3. Authorize a Payment (Command)

[source,bash]
----
curl -X POST http://localhost:8080/cards/{cardId}/authorize \
  -H "Content-Type: application/json" \
  -d '{
    "amount": 200
  }'
----

=== 4. Check Transactions (Query)

[source,bash]
----
curl http://localhost:8080/cards/{cardId}/transactions
----
