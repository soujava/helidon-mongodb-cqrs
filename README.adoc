= helidon-mongodb-cqrs

Sample Helidon MP project that includes multiple REST operations.

== Build and run


With JDK21

[source,bash]
----
mvn package
java -jar target/helidon-mongodb-events.jar
----

== Exercise the application

Basic:

[source, bash]
----
curl -X GET http://localhost:8080/simple-greet
Hello World!
----
